The Dude with a Thousand Faces the template

The Template is an outline for a generic monomyth. Replace each {variable} to create an outline for a 120-page screenplay.

The template is partitioned into acts, measures, and beats:

  • A beat is a sequence of one or more scenes. Each beat reveals information relevant to the story's core conflict.
  • A measure is a sequence of one or more beats. Each measure changes the hero's relationship to the story's otherworld.
  • An act is a sequence of one or more measures. Each act changes the hero's relationship to his/her/their homeworld.

See the FAQ below for more details.

Frequently Asked Questions

Who put you in charge?
Nobody. The authors do not enforce any of these rules.
Can beats occur off-screen?
Yes, but only if those beats are unambiguously implied by action or dialog which occurs on-screen.
What if my story has multiple heroes?
If a film features an ensemble cast, then any combination of one or more protagonists can act as hero during a beat.
Is that all?
Aye, all No. Following the template is necessary but not sufficient to produce a conventional screenplay. Feature-length films typically include many other noteworthy beats.
